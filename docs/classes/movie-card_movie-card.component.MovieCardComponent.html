    Class MovieCardComponent

    Component for displaying movie cards with detailed information, and favorite toggle functionality.

    The MovieCard component contains logic for displaying information about movie contained in the database. The component includes logic for displaying all movies in the database, filtering to only show movies included in a user's favorites list, and adding/removing movies from a user's favorites.
